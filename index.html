About project

The Graph Generator project is a new open source project which manages generating of graphs of various types. These graphs can be used as an input data for testing your graph algorithms.

Graph generator generates graphs according to its user wishes, e.g. complete graphs, discrete graphs, trees, simple acyclic graphs, etc.

The output file format is optional. Graph generator offers well-known formats for graph representation - Trivial Graph Format, Graph Modelling Language and DOT for graph visualization in Graphviz software.
